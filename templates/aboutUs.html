InfiniHacks Submission

Alex Li, Lasya Erukulla and Vishrutha Gopa

With the increasing number of diseases, there is a greater need for rapid diagnosis and early precaution. This is where MedCare can help. MedCare is a web application that collects data on the symptoms that a user is experiencing. The self-diagnosis tool provides a range of diagnoses based on the user's experiences using sophisticated computer algorithms. Not only does MedCare provide you with an early prediction, but it can also provide your doctor with new diagnostic options. MedCare can help you save a lot of time and mental stress.
